# ⚡ ExtremeGym - A Nova Era Fitness

Bem-vindo à **ExtremeGym**, uma plataforma fitness de alta performance com uma estética imersiva baseada em **Cyberpunk, Synthwave e Neon**. Este projeto combina o visual nostálgico dos anos 80 com a tecnologia de 2026 para oferecer uma experiência única aos usuários.

## 🚀 Funcionalidades Principais

- **Visual Imersivo**: Design totalmente customizado com CSS vanilla, utilizando efeitos de Glassmorphism, Neon Glow e Scanlines.
- **Scroll Snapping**: Navegação suave e magnética que centraliza cada seção perfeitamente na tela.
- **Bento Grid (Quem Somos)**: Conteúdo organizado em uma grade moderna com cards interativos.
- **Carrossel de Aulas**: Interface fluida para visualizar as modalidades disponíveis (Funcional, Lutas, Pilates, etc.).
- **Mapa Dinâmico de Unidades**: Localizador de academias com marcadores interativos e detecção de proximidade.
- **Área Administrativa**: Painel completo para gestão de equipamentos e catálogo de aulas.
- **Dashboard do Aluno**: Espaço personalizado para acompanhamento de treinos e progresso.
- **Social Feed**: Integração visual com postagens sociais para engajamento da comunidade.
- **Rodapé Premium**: Estrutura profissional de 4 colunas com links de navegação, endereços e QR code.

## 🛠️ Tecnologias Utilizadas

- **Core**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Estilização**: CSS Vanilla (Variáveis CSS, Flexbox, Grid)
- **Animações**: Framer Motion & CSS Keyframes
- **Carrossel**: Swiper.js
- **Persistência**: Local Storage Services (Simulação de Banco de Dados)
- **Icons & Fontes**: Orbitron, Inter, Google Fonts

## 📦 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/GabrielAron/ExtremeNeonGym.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
